# 간단 board crud 설계 프로젝트입니다.

## 개요

간단 게시판 ERD 설계 후 JDBC를 통한 DAO 구성을 통해 간단한 CRUD를 해볼 예정입니다.
thymeleaf를 이용하여 view 단을 구성할 예정이며 spring의 controller, service ,DAO 단 구성을 학습할 예정입니다.
